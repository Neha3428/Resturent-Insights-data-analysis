# Resturent-Insights-data-analysis
## Project Overview

This project analyzes restaurant data to extract valuable business insights related to customer preferences, restaurant ratings, pricing, cuisines, and service availability. The analysis was performed using Python in Jupyter Notebook as part of a Data Analytics internship task.

The goal was to clean, explore, visualize, and interpret restaurant data to support data-driven decision-making.
## Tools & Technologies
Python
Jupyter Notebook
Pandas
Matplotlib
## Project Workflow
Data Cleaning
Exploratory Data Analysis (EDA)
Data Visualization
Business Insights
## Project Outcome

Successfully transformed raw restaurant data into meaningful business insights through statistical analysis and visualizations, helping stakeholders understand customer behavior and restaurant performance.
